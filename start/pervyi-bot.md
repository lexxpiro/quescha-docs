# Первый бот



Чтобы создать вашего первого бота нужно создать канал, через который бот будет работать, а также саму программную логику его работы - автоворонку.

{% hint style="info" %}
Если вы хотите, чтобы бот работал на лендинге, то необходимо создать лендинг и добавить туда блок чат-бота. Таким образом, лендинг "превращается" не в просто лендинг, а в бот-лендинг;)
{% endhint %}

Добавим в каналы бота Telegram и создадим автоворонку, где бот спросит ваше имя и вернет его вам обратно.
