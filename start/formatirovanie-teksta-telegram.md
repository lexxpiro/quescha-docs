# Форматирование текста (Телеграм)

Используйте HTML теги для форматирования текстов

```
<b>Жирный</b>
```

```
<strong>Жирный</strong>
```

```
<i>Курсив</i>, <em>Курсив</em>
```

```
<u>Подчеркнутый</u>, <ins>Подчеркнутый</ins>
```

```
<s>Зачеркнутый</s>, <strike>Зачеркнутый</strike>, <del>Зачеркнутый</del>
```

```
<span class="tg-spoiler">Спойлер</span>, <tg-spoiler>Спойлер</tg-spoiler>
```

```
<b>bold <i>italic bold <s>italic bold strikethrough <span class="tg-spoiler">italic bold strikethrough spoiler</span></s> <u>underline italic bold</u></i> bold</b>
```

```
<a href="http://www.example.com/">Ссылка</a>
```

```
<a href="tg://user?id=123456789">Ссылка на пользователя</a>
```

```
<tg-emoji emoji-id="5368324170671202286"></tg-emoji>
```

```
<code>Код</code>
```

```
<pre>pre-formatted fixed-width code block</pre>
```

```
<pre><code class="language-python">pre-formatted fixed-width code block written in the Python programming language</code></pre>
```

```
<blockquote>Цитата</blockquote>
```

```
<blockquote expandable>Длинная цитата с кнопкой "свернуть/развернуть"</blockquote>
```
