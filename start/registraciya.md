# Регистрация

Зарегистрироваться в конструкторе вы сможете по ссылке [app.quescha.com/register](https://app.quescha.com/register)

{% embed url="https://youtu.be/B684FPn3aRM" %}

{% hint style="info" %}
Не забудьте подтвердить свой аккаунт в течении суток, иначе он будет удален
{% endhint %}

{% hint style="info" %}
При регистрации через бота, вы можете подтвердить свою регистрацию в любом боте на Квесча
{% endhint %}

{% hint style="info" %}
Если по каким-то причинам вы не смогли подтвердить аккаунт, то его можно подтвердить пополнив баланс на любую сумму
{% endhint %}
