# Переменные

Переменная является центральным звеном сценариев действий, которая позволяет передавать данные между блоками "действий" и возвращать данные в сценарий чат-бота.

Переменная имеет вид **#{название переменной}**

Переменная действует в рамках всех сценариев действий, подключенных к одному сценарному сообщению. Для передачи данных между сценариями действий также можно использовать Списки, Глобальные переменные, Google Таблицы, Обмен данными через серверы. Также данные между сценариями можно передавать в кнопках.

Переменная перестает храниться при выходе из сценария (не сценария действий!). Также переменные исчезают при запуске других цепочек сообщений.

Сама переменная может быть как строкой, так и массивом. В массив переменная превращается при сохранении в нее данных выборки из списков или гугл таблиц. Если пользователь может несколько раз вызывать сценарий, где данные накапливаются в переменную, позаботьтесь о том, чтобы очищать переменную перед такой операцией выборки. При сохранении отбора в переменные они очищаются по-умолчанию. Также очистить переменную можно присвоив ей пустое значение.

{% hint style="info" %}
Чтобы проверить переменную на существование, достаточно в условии написать **#{name}!=**

Выполнение данного условия означает, что переменная **name** существует
{% endhint %}
