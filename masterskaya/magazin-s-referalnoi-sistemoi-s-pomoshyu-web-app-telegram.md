# Магазин с реферальной системой с помощью Web App Телеграм

Многоуровневая реферальная система позволяет совершать покупки в магазине, начислять бонусы согласно маркетинг плану, предоставлять скидки на покупку товаров, отправлять сообщения своим рефералам.

Если ваша система разрастется до такой степени, что появятся рефералы уровня, которого нет в списке Маркетинговый план, то значения бонусов и скидок будут браться из последнего уровня в списке.

Как настроить реферальный магазин под себя смотрите на видео на нашем Ютуб канале.
