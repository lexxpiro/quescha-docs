# Начисление баллов рефереру

Архив с ботом, где начисляются баллы рефереру при переходе по реферальной ссылке. Бот носит обучающий характер. Необходимо подключить бот к цепочке сообщений и перейти по ссылке https://t.me/\[ваш бот]?start=1



{% hint style="info" %}
Пример использует реферальную систему, использующую списки и не имеет отношение к реферальной системе встроенной в конструктор
{% endhint %}
