# Отправка сообщения первым в WhatsApp

Ни для кого не секрет, что в мессенджере WhatsApp можно написать первым. Бот WhatsApp не исключение. Как это сделать?

Вам потребуется создать Список с перечнем номеров ваших клиентов. Номера должны быть вида 7xxxxxxxxxx. Также создайте поля для мессенджера, где укажите значение wa, и поле для идентификатора вашего бота, где в качестве значения укажите BotID вашего бота (номер телефона).

У вас уже должен быть подключен ваш бот в WhatsApp с вашей автоворонкой.

Теперь создайте цепочку сообщений и сценарий для рассылки по списку номеров. В сценарии действий вы читаете список ваших номеров и сохраняете все значения в переменные (3 переменных).

В секции Отправки в бот создайте ваше сообщение и в качестве адресата укажите ваши переменные в полях ClientID, Messenger, BotID.

Вам останется лишь запустить вашего бота и сообщение будет разослано.

{% hint style="info" %}
Будьте аккуратны, в случае **спам-рассылок** ваш номер может быть заблокирован в WhatsApp
{% endhint %}
