# Передача начальной информации в бот

В сервисе существует возможность передать начальную информацию в бот

### Telegram

Передать данные можно в параметре start в ссылке на бот https://t.me/myservice\_bot?start=111

Данные будут сохранены в переменную start и могут быть использованы в сценарии действий.

### Бот-лендинг

Передать начальную информацию можно через параметры в ссылке на бот-лендинг. Все параметры с их значениями будут переданы в переменные. Например при заходе на бот-лендинг по ссылке subdomain.quescha.com?param1=1\&param2=2\&param3=3, будут созданы переменные param1 со значением 1, param2 со значение 2, param3 со значением 3

{% hint style="info" %}
В другие мессенджеры начальную информацию передать нельзя
{% endhint %}
