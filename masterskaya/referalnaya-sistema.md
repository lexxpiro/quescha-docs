# Реферальная система

Готовый сценарий реферальной системы, которая позволяет регистрировать реферала по ссылке, получать реферальную ссылку, смотреть баланс, листать список рефералов, отправлять сообщение своей реферальной сети.

После импорта подключите канал к цепочке и замените ссылку на бота в сценарии действий "Получение реферальной ссылки"
