# Регулярные выражения

#### Проверка номера телефона

```
^((8|\+7)[\- ]?)?(\(?\d{3}\)?[\- ]?)?[\d\- ]{10,10}$
```

#### Проверка даты

```
^(\d{1,2})[.\s-/](\d{1,2})[.\s-/](\d{4})$
```

#### Проверка Email

```
^[-\w.]+@([A-z0-9][-A-z0-9]+\.)+[A-z]{2,4}$
```

#### Проверка времени в формате **HH:MM:SS**

```
^([0-1]\d|2[0-3])(:[0-5]\d){2}$
```

#### **Проверка ФИО (ФИ)**

```
^[а-яА-ЯёЁa-zA-Z]+ [а-яА-ЯёЁa-zA-Z]+ ?[а-яА-ЯёЁa-zA-Z]+$
```

#### **Цифра от 1 до 5**

```
^[1-5]{1}$
```

#### Проверка отправки геопозиции в Телеграме

```
^\d+\.\d+:\d+\.\d+$
```

#### Строка от 1 до 15 символов

```
^.{1,15}$
```
