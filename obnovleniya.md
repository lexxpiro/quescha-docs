## 31.10.2025

* Добавили параметры, который можно добавлять к ссылке приложения календаря. Если требуется ограничить количество дней записи от текущей даты, то в ссылке приложения передайте GET параметр freedays с количеством дней (напр. freedays=30). Если требуется сделать доступными часы (четные/нечетные), то используйте GET параметр freetime. Возможные значения: even - доступны четные часы, odd - доступны нечетные часы. Также для freetime. Укажите значение в формате "workers\_n\_m\_k". n - количество работников, m - интервал в часах, k - номер работника. Также для freetime. Укажите значение в формате "x\_y". x - интервал занятых часов, y - интервал свободных часов, в часах. Также для freetime. Укажите число часов x, которые будет занято. Следующий час будет свободным. Эквивалентно "x\_1"
* Добавлен API метод "Получение записей приложения календарь"
* В настройках вебап приложения календаря добавлена кнопка генерации времени расписания с заданным шагом в минутах
