# Группы

Если вы используете платформу Квесча для нескольких проектов, то для удобства работы создана возможность группировки элементов.&#x20;

Группы позволяют отображать только те элементы, работа с которыми происходит в данных момент.

Группируются:

* каналы
* бот-лендинги
* автоворонки
* магазины
* записи на услуги
* сценарии действий
* списки
* глобальные переменные
* сотрудники
* платежные системы
* CRM
* Google-аккаунты
* Яндекс-аккаунты

Хорошая практика разработки - сначала создаете группу, выбираете ее, чтобы другие элементы вас не отвлекали и приступаете к разработке. Все создаваемые сущности будут созданные в группе.

Группы можно скопировать. Будут скопированы: лендинги, цепочки автоворонок, сценарии записи, магазины и сценарии действий. После копирования, вам потребуется заново привязать каналы
{% endhint %}
