# Ошибки

{% code overflow="wrap" %}
```
401 - ошибка авторизации. Убедитесь, что вы передаете заголовок Authorization со значением вашего API ключа

422 - ошибка в параметре. Убедитесь, что вы передаете обязательные параметры метода

429 - частота запросов. Превышена частота запросов к API

409 - запрет сервера. Возможная причина, например, превышен лимит на количество товаров в каталоге.
```
{% endcode %}
