# Каналы (Боты)

В каналы добавляются боты, которых вы создали в мессенджерах. При добавлении бота отобразится подробная инструкция создания бота в выбранном мессенджере.

В конструкторе можно создавать ботов:

* Телеграм
* ВК
* WhatsApp
* Viber
* На сайте
