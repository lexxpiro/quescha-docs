# Особенность WhatsApp

С мая 2023 года в WhatsApp перестали работать кнопки. Учтите это при создании сценария бота для этого мессенджера.

Рекомендуем вместо отправки кнопок использовать варианты выбора текстом обычного сообщения, ответ на который будет обработан далее, как выбранный вариант. Например:

Выберете действие:&#x20;

1 - действие 1&#x20;

2 - действие 2

3 - действие 3

После отправки выбранного номера подписчиком, этот ответ можно обработать в условиях, например такими выражениями:&#x20;

Выражение в условии 1: \[Answer] = 1

Выражение в условии 2: \[Answer] = 2

Выражение в условии 3: \[Answer] = 3
