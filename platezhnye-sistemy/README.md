# Платежные системы

Для подключения платежной системы, вы должны иметь аккаунт этой платежной системы. Далее добавьте интеграцию на странице Платежные системы в сервисе Квесча и введите необходимые данные для совершения и обработки платежа.&#x20;

{% hint style="info" %}
Там, где это требуется, не забудьте установить адрес для уведомлений в самой платежной системе, чтобы ваш чат-бот смог обрабатывать факт совершения платежа
{% endhint %}

<figure><img src="../.gitbook/assets/Платежные-системы (2) (1).png" alt=""><figcaption></figcaption></figure>

Далее вся настройка работы происходит в сценариях действий, который вы запускаете в блоке до его запуска или раньше.&#x20;

В сценарии действий необходимо сформировать платежную ссылку и вывести ее клиенту.

Для приема платежа включите "Ожидание платежа" в блоке, который запустится при поступлении платежа. Для дополнительной обработки, вы можете настроить прием платежа в сценарии действий. Не забудьте подключить его в сообщении "ожидания платежа" до запуска блока и добавить действие приема платежа

{% hint style="info" %}
Если у вас не получается переходить по платежной ссылке через кнопку, то скорее всего платежная система возвращает какую-либо ошибку при получении ссылки. Чтобы ее увидеть - выводите платежную ссылку в самом сообщении, а саму кнопку временно удалите
{% endhint %}
