# Настройки аккаунта

<figure><img src=".gitbook/assets/Настройки-аккаунта (1).png" alt=""><figcaption></figcaption></figure>

В настройках аккаунты вы можете:

* задать часовой пояс работы бота
* сменить пароль своего аккаунта
* задать API ключ для работы с [API конструктора](api/)
* производить экспорт/импорт своих ботов
* выдать доступ для совместного использования аккаунта
