# Массовые операции

В конструкторе существует возможность проводить операции над сущностями в CRM массово.

Для этого, вам потребуется сделать выборку из списка и поместить эту выборку в переменные. Таким образом, переменные содержат не одно, а множество значений, по факту являясь массивами. Например, вы можете поместить в переменную выборку по ID сделкам, которые хотите изменить.

Работу с массовыми операциями поддерживают не все операции с CRM. Вот перечень поддерживаемых операций:

#### AmoCRM

* Добавить примечание к сделке
* Добавить примечание к контакту
* Изменить сделку
* Изменить контакт
* Изменить компанию

#### Битрикс24

* Добавить комментарий
* Изменить сделку
* Изменить лид
* Изменить контакт
* Изменить компанию
