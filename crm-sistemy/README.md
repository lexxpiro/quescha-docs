# CRM системы

В сервисе Квесча реализована интеграция с amoCRM и Bitrix24. Добавьте интеграцию через страницу "CRM системы". Далее вся настройка осуществляется в сценарии действий.

{% hint style="info" %}
В amoCRM имеется виджет, для отправки сообщений клиенту прямо из карточки сделки и контакта
{% endhint %}

{% hint style="info" %}
Для установки интеграции Битрикс24, сначала установите приложение в маркете аккаунта Битрикс24
{% endhint %}
