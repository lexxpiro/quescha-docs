# Бот-лендинги

Конструктор Квесча позволяет создавать лендинги и добавлять к нему блок чат-бота. При создании лендинга, он добавляется в список каналов для привязки цепочек сообщений, записей на услуги и магазинов.

{% hint style="info" %}
При создании лендинга блоки Текст и Заголовок могут содержать код html разметки, что значительно расширяет возможности верстки. Например, вы можете использовать готовые компоненты css-фреймворков, таких как Tailwind CSS, чтобы делать лендинги любой сложности.
{% endhint %}

{% hint style="info" %}
В блоках Текст и Заголовок можно использовать иконки Font Awesome в коде
{% endhint %}

Особенностью бот-лендинга как канала связи заключается в том, что мы не знаем когда посетитель уйдет с сайта. Строить автоворонку для бот-лендинга нужно с учетом этого факта. Слать периодические сообщения или сообщения с задержкой не имеет особого смысла, так как скорее всего этих сообщений посетитель не получит. Все взаимодействие должно строиться по принципу запрос-ответ.
