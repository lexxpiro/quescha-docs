# ChatGPT

В конструкторе есть 3 вида нейросети

1. Completion - текстовый запрос в нейросеть с получением ответа от нее. Если вы используете свой аккаунт openai.com, то сможете указать любую доступную модель.
2. Assistant - работа с ассистентом openai позволит вам обучить нейросеть на основании ваших собственных данных. Работает только на ваших аккаунтах openai.com.
3. DALL-E - генерация изображений. Работает только на ваших аккаунтах openai.com.
