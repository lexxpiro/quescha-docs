# Списки/Глобальные переменные

#### Списки

Списки это таблицы данных, которые используются для работы с чат-ботами. Вы можете создать поля таблицы вручную, либо загрузить файл excel с подготовленной шапкой полей.

{% hint style="info" %}
Поля после создания нельзя будет отредактировать. Если вам потребуется дополнительные поля, продумайте это заранее, либо вам придется перезалить таблицу из файла с новыми полями.
{% endhint %}

Любая запись списка содержит автоматически заполняемые поля ClientID, Messenger, BotID. Эти поля - своеобразный след того подписчика, кто эту запись создал. По этим данным, вы можете находить в списке данные подписчика бота, используя шаблоны \[ClientID], \[Messenger], \[BotID].

#### Глобальные переменные

Глобальные переменные используются для хранения данных при работе с чат-ботами (например, порядковый номер договоров). Глобальные переменные являются общими для всех подписчиков.
