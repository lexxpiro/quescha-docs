# Тарифы

{% hint style="info" %}
**Пробный тариф:** первые 7 дней полной функциональности до 100 подписчиков
{% endhint %}

{% hint style="info" %}
**Бесплатный тариф:** бессрочно тариф Старт с некоторыми ограничениями до 100 подписчиков&#x20;
{% endhint %}

{% hint style="info" %}
Если на используемом тарифе число подписчиков превысит указанный лимит, то ваши боты продолжат работать. Однако, учтите, что превышение не может быть больше лимита подписчиков следующего за вашим тарифом. Вам необходимо будет выбрать новый тариф до момента списания платы.
{% endhint %}

Скриншот страницы тарифов https://app.quescha.com/payments/tariffs
